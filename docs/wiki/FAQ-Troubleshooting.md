# FAQ & Troubleshooting
**Stand:** 2026-03-15

## FAQ

### Is wiki content the source of truth?
No. The source of truth remains the repository docs under `docs/`.

### Which checks should I run before proposing changes?
Run at least `make verify`. For benchmark-related work, also run
`make stats-benchmark`.

### Why are many detailed docs still in German?
The project uses an incremental public-readiness strategy: English entry points
first, deep historical/technical detail can stay German-first for now.
