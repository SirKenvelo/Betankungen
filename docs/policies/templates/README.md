# Tracker Templates
**Stand:** 2026-03-13

Diese Vorlagen sind die kanonische Basis fuer neue Tracker-Eintraege gemaess
`POL-001`.

## Vorlagen

- `issue.template.md`
- `backlog-item.template.md`
- `task.template.md`

## Nutzung

1. Vorlage kopieren.
2. Frontmatter komplett ausfuellen.
3. `created` und `updated` auf das aktuelle Datum setzen (`YYYY-MM-DD`).
4. ID-Regeln aus `POL-001` einhalten (neu: `ADR/BL/ISS/TSK` 4-stellig).

## Dateinamenskonvention (Empfehlung)

- Issue: `docs/issues/ISS-0001-kurzer-slug/issue.md`
- Backlog: `docs/backlog/BL-0001-kurzer-slug/item.md`
- Task: `docs/backlog/BL-0001-kurzer-slug/tasks/TSK-0001-kurzer-slug.md`
