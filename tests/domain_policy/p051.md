# P051
**Stand:** 2026-04-07

Gap-Flag-Design-Policy: Ohne expliziten Ausnahme-Opt-in `--missed-previous`
darf `missed_previous` nie automatisch gesetzt werden; normale kurze
Distanzen duerfen keinen `P-050`-Standardprompt mehr erzeugen.

Golden-Template-Dateien:
- `fixtures/p051_base.sql`
- `cases/t_p051__01__no_auto_gap_flag_without_confirm.sh`

Erwartung fuer Cases:
- Ohne `--missed-previous` bleibt `missed_previous=0`; der normale
  Kurzdistanz-Fall wird ohne `P-050`-Rueckfrage gespeichert.
